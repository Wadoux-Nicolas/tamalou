# Installation du projet :

1. Installer Python 3.12

2. Créer un environnement virtuel :
   ```shell
   python -m venv venv
   ```

3. Activer l'environnement virtuel :
   ```shell
   source ./venv/bin/activate
   ```

4. Installer les dépendances :
   ```shell
   pip install -r requirements.txt
   ```

5. Lancer le projet :
   ```shell
   fastapi dev main.py
   ```
